# FontForge

**Stylish Unicode Font Generator & Symbol Library**

A sleek, dark-themed web tool for generating stylish Unicode fonts and copying special symbols — built for creators, designers, and social media enthusiasts.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit-6ee7f7?style=flat-square)](https://fontforge-app.netlify.app)
[![Made by](https://img.shields.io/badge/Made%20by-Md.%20Mainul%20Islam-a78bfa?style=flat-square)](https://github.com/M41NUL)
[![License](https://img.shields.io/badge/License-MIT-f9a8d4?style=flat-square)](LICENSE)

</div>

## ✨ Features

- **50+ Font Styles** — Bold, Italic, Script, Fraktur, Double Struck, Monospace, Bubble, Glitch and more
- **700+ Symbols** — Stars, Arrows, Decorative, Math, Currency, PC, Gaming, Apple & Brand, Aesthetic, Invisible
- **Live Preview** — Type anything and see all font styles update instantly
- **One-Click Copy** — Copy any font or symbol to clipboard instantly
- **Favorites System** — Save your favorite fonts with localStorage persistence
- **Symbol Search** — Search across all symbol categories
- **Dark UI** — Minimal dark design with glassmorphism cards
- **Mobile Friendly** — Fully responsive on all screen sizes

## 🚀 Getting Started

No installation needed. Just open `index.html` in any browser.

```bash
git clone https://github.com/M41NUL/FontForge.git
cd FontForge
# Open index.html in browser
```

Or simply visit the **[Live Demo](https://fontforge-app.netlify.app)**.

---

## 🛠️ Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JS** — zero dependencies
- **Unicode** character maps for font transforms
- **Google Fonts** — Space Grotesk + Playfair Display
- **localStorage** for favorites persistence
- **CSS backdrop-filter** glassmorphism UI
